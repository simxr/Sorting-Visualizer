# Sorting-Visualizer

## A simple frontend implementation to visualize various sorting algorithms including:
1. Bubble sort_<br>
2. Insertion sort_<br>
3. Selection sort_<br>
4. Quick sort_<br>
5. Merge sort_<br>
6. _Radix LSD sort_<br>
7. _Heap sort_<br>

### Includes
 + Toggles to select _Delay_ and _Array Size_
 + Copy-able codes
